# Ball Drop

An Arduino Uno game on a 8 x 8 LED Matrix

## Components required

- Arduino UNO

- 2 x Breadboards

- Red 8x8 LED matrix

- 1 x Max7219 driver

- LCD 1602 Module

- 10 μF, 0.1 μF capacitors

- 10k, 1k, 220 Ohm resistors

- Joystick Arduino Module
